주요 기능
★ 말벗
- 말벗 목소리 설정 가능 (일반 TTS, 가족) 목소리 학습. 가족으로 커스터마이징했을 경우 말투도 그에 맞게.

1. 객체 인식, 얼굴 스캔 감정 인식해서 감정에 맞는 말을 건냄 
	사용자가 퇴근 후 집으로 돌아옴 -> 들어온것을 인식해서 로봇이 다가온 후 얼굴 스캔. -> 감정을 인식 후 대화를 시작 지침, 우울, 등 부정적 -> (오늘 많이 힘들었어?,  수고 많았어. 등)
	무표정, 인식안됨 -> 잘 다녀왔어? 어땠어?
	행복, -> 좋은일 있었어? 
	그 후 이어지는 대화는 챗 gpt 사용.
	
3. 점등/소등
	사용자는 침대에 누워서 핸드폰 중. 이제 자고 싶은데 소등하러 다시 일어나기엔 많이 귀찮음. 
	불꺼달라고 부탁 => 인식 후 모든 전등을 소등해줌 
	- 밤에 화장실 가고 싶어서 눈이 떠짐. -> 화장실 가게 불을 켜달라고 부탁 -> 인식 후 불을 켜줌 (방 + 화장실)

4. 음식 레시피 추천 (우선순위 : 최하)
	사용자가 원하는 재료로 어떤어떤요리(매콤, 담백, 국, 볶음 등)를 하려고 하는데 어떻게 해야 하냐 물어봄
	원하는 종류의 음식을 토대로 음식을 음식이름을 추천해줌. 사용자가 선택을 한다면 선택된 음식의 레시피를 읽어줌.
	앱으로도 레시피 띄워줌

6. 아침에 깨워주기 
	아침에 일어나야 할 시간에 사용자가 일어나 있는지 자고 있는지 확인 -> 일어났으면 아침인사. 
	-> 계속 자는 중이라면 불을 켜고 깨워주기 (일어날 때까지)


★ 코디 추천
- 핸드폰으로 사용자의 옷을 미리 등록.
- 코디평가 : 옷 어때? ==> 점수로 평가. (패션잡지등 학습에 가까울수록 100점) or 30점 이하 점수면 다른옷 추천 (70~)
- 코디추천 : 오늘 뭐입지? ==> 다른 아무런 추가 요청이 없음 -> 일상복 추천 (날씨 기준)
- 코디추천2 : 오늘 면접 있는데 옷 추천해줘 ==> 면접에 입고 가기 좋은 옷 추천
- 코디추천3(우선순위 최하) : 이 바지에 맞는 옷 뭘 살까?  ==> ex) 파란색 계통, ~~ 옷 사면 좋을거 같아. 예상도 사진 보여줌.


★ 앱 기능

- 친구와 채팅 
	- 심심이 같은 기능. 
	- 채팅은 학습돼서 퇴근후 대화할때 대화가 이어짐
- 옷 등록
- 코디 추천 보여주기 (2d 사진)
- 옷 구매시 예상도 보여주기

- 추천받은 음식레시피 리스트
- 설정 
	- 보호자 입력
	- TTS 설정 (기본, 커스텀목소리)
	- 
- 같은 제품 사용자끼리 커뮤니티 (익명)

★ 알림
아침에 사용자의 움직임이 인식되면 오늘 날씨를 알려줌
먹어야 할 약이 있다거다 매일 해야하는 일이 있다면 미리 알려줌
일정 시간동안 움직임이 없거나 나타나지 않는 경우 사용자를 찾아가서 상태를 확인함 → 위험이 인지된 경우라면 보안파트로 넘어감 or 다시 잠들거나 위험 없다고 판단되면 돌아감

★ 보안
사용자가 넘어짐 → 큰 소리를 인지하고 봇이 가까이 와서 괜찮은지, 가족에게 연락할 지 물어봄 → 연락해달라고 가족에게 알림을 보냄 or 대답을 못한다면 두 번 더 물어본 후 그럼에도 대답이 없으면 119에 신고함
